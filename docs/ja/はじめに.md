# はじめに

このリポジトリをWindowsにcloneし、Codexで開き、研究したいことを曖昧なまま伝えます。Codexは環境を先に調べ、ログイン・ブラウザ・予算など人間にしか決められないことだけを、分かりやすい日本語で一つずつ確認します。

研究開始後、人間は細かなコードを判断しません。GitHub ProjectとCodex Appで次を確認します。

- 誰が何をしているか
- 研究が進んでいるか
- 何が新しく分かったか
- 何時間・何GPU時間・何円使ったか
- 次に何をするか
- 研究ハーネスが正常か

研究計画と実験の判断はResearch Plannerと、設定済みruntime profileから解決されたResearch Executorが、人間所有のMission・データ・法的境界・ハード予算・公開方針の内側で自律的に行います。
