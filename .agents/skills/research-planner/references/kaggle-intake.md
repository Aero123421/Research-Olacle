# Kaggle Planner intake

Before issuing the first Campaign, establish a dated Evidence Pack containing:

- official Overview, Data, Evaluation, Rules, Timeline, and code requirements
- metric implementation and direction
- submission mode and public/private leaderboard risks
- file/table inventory, identifiers, joins, target, missingness, duplicates,
  time/group/entity structure, train/test shift, and leakage candidates
- a reproducible baseline, fold behavior, runtime/compute profile, and sample
  submission/order checks
- domain and real-world data-generating process
- analogous competitions, papers, negative results, Discussions, public
  notebooks, and current community evidence

EDA is decision-oriented. Stop analyses that cannot change the next strategic
decision.
