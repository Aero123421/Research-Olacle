# Chrome runbook

- Install and connect the official Chrome plugin from Codex/Work Plugins.
- Use a dedicated Chrome profile for research automation when practical.
- Confirm the extension says Connected in the active profile.
- Keep only one automation task controlling that profile at a time.
- Stay signed in to ChatGPT; human handles login/MFA.
- If disconnected, remove/re-add the plugin or restart Chrome and the desktop app
  before changing browser mode.
