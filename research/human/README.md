# Human comprehension layer

Research rigor is independent of human explanation level. These files help the
Director translate durable research state into concise, plain-language updates.

- `research/setup/HUMAN_PROFILE.md`: stable communication preferences
- `CONCEPT_MEMORY.md`: concepts already explained
- `research/CURRENT_BRIEF.md`: regenerated current snapshot
- `reports/visuals/`: deterministic diagrams and charts
