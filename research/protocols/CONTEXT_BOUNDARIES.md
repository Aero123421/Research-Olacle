# Context boundaries

## Planner receives

Mission, current strategy, bounded memory, research landscape, evidence index,
selected Campaign Handoffs, compute/deadline state, and the current ResearchPlan
evidence pack.

## Executor receives

One Campaign Contract, its Context Pack, evaluation contract, starting commit,
compute policy, and only relevant prior evidence.

## Advisors receive

One Question Pack, minimum supporting evidence, expected output, and no unrelated
history.

## Never inject by default

Full conversations, raw terminal logs, all notebooks, every paper, unrelated
campaign details, credentials, or private raw data.
